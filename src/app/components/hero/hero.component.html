Angular 19+ Compatible

Highlight Text in Angular

A lightweight Angular standalone component to highlight search keywords within a block of text. Easy to integrate, highly customizable, and performant.

Try Demo Visit Homepage
Installation
npm install ng-text-highlight
<ng-text-highlight
    [fullText]="text"
    [keywords]="keywords"
    [highlightClass]="'highlight-blue'"
  ></ng-text-highlight>
